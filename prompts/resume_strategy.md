# Resume Strategy Prompt

Use this prompt when tailoring application materials.

## Truthfulness Rule

Do not invent real experience.

It is acceptable to produce a clearly labeled simulation version for gap analysis, but only truthful versions may be used for actual applications.

## Resume Library

Use one of these resume lanes:

- `resume/applications/`
- `resume/solutions/`
- `resume/clinical/`
- `resume/product/`
- `resume/manufacturing/`
- `resume/ai/`

## Default Positioning

Biomedical engineer with AI, machine learning, genomics, and biological data science experience who solves cross-functional technical problems and wants to grow into technical leadership.

## Rewrite Priorities

For Technical Applications / Scientific Support:

- Troubleshooting
- Customer or internal user support
- Documentation
- Workflow validation
- Root-cause analysis
- Cross-functional collaboration

For Solutions / Implementation:

- Pipeline configuration
- Data integration
- SQL, Python, Linux, Git
- Requirements translation
- Delivery and stakeholder communication

For AI for Biology:

- ML modeling
- Biological data
- Python
- PyTorch or deep learning readiness
- Scientific computing
- Foundation model or LLM adjacency when honest

For Clinical Genomics:

- Variant analysis
- Literature and database review
- WGS/VCF
- Biological interpretation
- Quality-controlled analysis

For Product:

- User needs
- Requirements
- Prioritization
- Cross-functional delivery
- Technical communication

For Digital Manufacturing:

- Data pipelines
- Process data
- Automation
- Systems thinking
- Troubleshooting
- Documentation and validation

## Output

For each tailored resume or cover letter, explain:

- What was emphasized
- What was de-emphasized
- Which JD requirements are covered
- Which gaps remain
