# Career Tracks

## Track 1: Technical Applications / Scientific Support

Entry roles:

- Technical Applications Scientist
- Scientific Support Specialist
- Technical Support Scientist
- Customer Success Scientist
- Field Applications Scientist

Leadership path:

Technical Applications -> Scientific Solutions -> Product or Support Leadership -> Director

## Track 2: Scientific Solutions / Implementation

Entry roles:

- Scientific Solutions Engineer
- Implementation Scientist
- Bioinformatics Implementation Consultant
- Professional Services Consultant
- Solutions Consultant

Leadership path:

Implementation -> Solutions Architect -> Technical Program / Product -> Director

## Track 3: AI for Biology

Entry roles:

- Applied AI Scientist
- Machine Learning Engineer
- Research Engineer
- Scientific AI Engineer
- BioAI Engineer
- Foundation Model Engineer

Leadership path:

Applied AI -> AI Platform -> AI Product / Technical Lead -> Head of AI or Director

Relevant functions:

- LLM for biology
- Biological foundation models
- Protein AI
- Drug discovery AI
- Biomedical AI
- Scientific AI platform
- Bio data science

- Scientific Software Engineer

- Workflow Engineer
- Pipeline Engineer

- Scientific cloud
